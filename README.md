# ToDooooNE

トゥードゥーンは、Trelloライクなカンバン方式のTodo管理ツールです。

- Vue 2.6.11
- Vuex 3.4
- VueCLI 4.3.1

を利用して作成してます。
また、LocalStrage上でデータを保存してます。

※PCのみの対応です。（スマホやタブレットでは動作未確認のため）
